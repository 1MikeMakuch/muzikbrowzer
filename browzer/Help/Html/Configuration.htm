Configuration

  • General

    • Directories to scan for audio files. Tell muzikbrowzer where to look for mp3/ogg files on your computer or network. You don't want to enter "C:\" here, rather, enter the specific directory(ies) where your music files are stored.
    • Audio File extensions. Normally the default "mp3" is sufficient. Only modify this list if you have mp3 files with an extension other than "mp3". If you have ogg files, add "ogg" to this list.
    • Location of muzikbrowzer database. Normally the default location is copasetic. This is where muzikbrowzer stores the database, saved playlists, log files etc. If you have some reason to store the muzikbrowzer database and other data in a folder other than the default "Program Files\muzikbrowzer" you may do so.
    • Scan. Performs the scan and builds the muzikbrowzer internal database. If the database exists it is first deleted then completely rebuilt.
    • Scan for New Files. Retains the existing database and scans in new audio files. Use this when you've added new music files into the directories specified above or, you've added one or more directories to the list of directories to scan. Scan for New is significantly faster than a complete Scan because it retains the existing database and only searches for new files. Note that if you use id3 tag editing software other than muzikbrowzer, you'll have to perform a complete Scan in order for muzikbrowzer to pickup those changes.
    • Run at Startup. This will cause muzikbrowzer to run when you logon. Tip: see the Autolog program at www.sysinternals.com.

  • Irman

    • Irman COM Port. Choose the serial COM Port you have the Irman infrared receiver plugged into. Usually 1 or 2. Choose Disabled if you do not have one.
    • Initialize. Click on Initialize after you have chosen a COM Port to test muzikbrowzer communication with the Irman. On Success you should get an "Ok".
    • IR Pulses per press. You may have to adjust this value depending upon how your remote control works with the Irman. This value represents the number of ir code sequences transmitted each time you press a button on your remote control. It will vary with different remote controls. See the procedure below for determining this value.
    • Record. Click Record to teach muzikbrowzer your remote control codes. According to Irman most all infrared remote controls will work with the Irman.
      • Click Record.
      • Click the Up button on the Irman Configuration Dialog. The Code value should indicating reading....
      • Now press the button on your remote control which you choose to serve as the Up button. Upon receiving a signal the Code value should display a sequence of hexidecimal digits corresponding to the button pressed.
      • Description. Optional but suggested, enter a brief description of each remote control button. For example if you are using the numeric buttons as the Up/Down/Left/Right buttons you should enter "8" as the description for Up, "2" for Down, etc. These descriptions are displayed when you choose Help via the remote control menu Help button, to help you remember which remote control buttons do what.
      • Repeat for all remaining buttons on the dialog.
      Note: Apparently the volume buttons on many remote controls operate differently than most of the rest of the remote control buttons. Consequently you may find that the Irman does not pick up the volume buttons on your remote control consistently. If this happens, simply choose another pair of buttons to serve as Vol Up/Down, or you may choose to leave the volume on muzikbrowzer set to maximum and only use the volume on your stereo.
    • Test. When you have taught muzikbrowzer some or all of your remote control codes click on Test to verify that the Irman is correctly reading signals from your remote control and that each button learned is distinguished from the others. In some cases more than one button on a remote control results in the Irman producing the same hexidecimal sequence. In this case you may not be able to use those buttons or in the worst case you may have to try a different remote control.
    • Ir Pulses per press
      • First "teach" muzikbrowzer your remote control's ir code sequences by performing the "Record" procedure above.
      • Start with a value of 1 IR Pulses per press.
      • Click OK and save the configuration, and return to the muzikbrowzer screen.
      • Try a few different remote control buttons, up, down, left, right.
      • Observe what happens on the muzikbrowzer screen.
      • If you for instance you are observing 2 up's, in response to a single press of the up key, then try increasing the 'IR Pulses per press' value to 2.

  • Fonts & Colors.


    • muzikbrowzer is designed to accomodate the relative low resolution of tv screens. Adjust Font type, size and colors to your preference.

  • Password

    • Enter your registration and license password here.
    • Without a password muzikbrowzer will run in Trial Mode. You have 60 days to decide whether or not to purchase.
    • To purchase click on the Purchase link at www.muzikbrowzer.com.
    • At that time you'll be given your license password which will work on one pc only.
    • You must enter your name, email, and password exactly identical as when you entered it in the Purchase form.
    • Your HostId must match also. If it does not for some reason you should send an email to support@muzikbrowzer.com explaining why you need a new password.

  • Ogg Vorbis
    • To enable muzikbrowzer to play Ogg Vorbis (.ogg) files you'll need to install an Ogg Vorbis Direct Show filter. This one is known to work with muzikbrowzer; http://tobias.everwicked.com/. Just download and install it and muzikbrowzer will be able to play ogg files.
