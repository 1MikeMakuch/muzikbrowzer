Muzikbrowzer files

  • C:\Program Files\Muzikbrowzer default installation folder
  • MuzikbrowzerCachedDrives - Muzikbrowzer's Add File/Foder dialog caches your drive letters here.
  • Muzikbrowzer.chm - help file
  • Muzikbrowzer.exe - Muzikbrowzer executable
  • unins00*.* - uninstall files
  • Menu/Settings/General "Location of Muzikbrowzer database files" defaults to C:\Program Files\Muzikbrowzer. Changing this setting changes location of the following files;
    • Muzikbrowzer.folders - Folders you specify in Menu/Settings/General "Audio file folders"
    • Muzikbrowzer.log - Error/Debug log
    • Muzikbrowzer.mb, Muzikbrowzer.mbfls, Muzikbrowzer.mbtgs - Muzikbrowzer Library database
    • playlists - Folder in which your .m3u playlists are stored
    • skins - Folder in which Muzikbrowzer skins are stored. SkinDef.mbsd (txt file) stores default skin settings. SkinDefCustom.mbsd (txt file) stores skin settings modified by user via Settings/Display dialog.
